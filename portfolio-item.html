The Year 2020: The Game

By Patrick Celedio

In this game, you must survive against zombies and giant coronaviruses with your trusty AK-47.

> > > > >

This game was made using Game Maker Studio 2. The graphics were drawn using Aseprite. The sound effects and music are from ZapSplat. There are ten rooms to survive from the zombies and coronaviruses. The goal is to safely arrive to the store to get groceries and come back home in one piece!

This is my first attempt at making a game using Game Maker Studio 2! I chose to create a two-dimensional platform type shooting game as I am a fan of these types of games such as Contra, Metroid, Castlevania, or Alien Hominoid. Every graphic, sound, and movement has some lines of code behind it to create the behavior. What I learned from this is that the more an object exhibits complex behaviors, the more code is written behind the scenes. It made me think about big budget video games in real life and how cinematic or realistic the gameplay and characters are. I have concluded that so much time and effort can be used just for the simplest in movement, but complex in code.
